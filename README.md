# 🚚 Logistics Dashboard

---

## 📝 Introduction

This project is an interactive Logistics Dashboard developed using Power BI Desktop, designed to provide actionable insights into logistics operations including orders, deliveries, drivers, hubs, and vehicles.

The dashboard focuses on enabling stakeholders to monitor performance, identify inefficiencies, and make data-driven decisions in real-time.

## 🎯 Project Objective

The primary objective of this project is to:

- Analyze logistics performance across multiple dimensions
- Track delivery efficiency and customer satisfaction
- Monitor driver performance and delays
- Evaluate hub capacity utilization
- Understand vehicle usage and operational distribution

## 🛠️ Tools & Technologies Used

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
- Power Query (ETL)
- Data Visualization Techniques

## 📂 Dataset Descriptaion 

The dataset used in this project includes logistics-related data such as:

- Orders Data – Order count, delivery time, status
- Drivers Data – Driver details, experience, ratings, delays
- Hubs Data – Hub capacity, processing efficiency
- Vehicles Data – Vehicle types, models, usage, maintenance status

## 🔄 Project Overflow

### 1. Data Collection & Cleaning

- Imported raw data into Power BI
- Cleaned and transformed using Power Query
### 2. Data Modeling
- Created relationships between multiple tables
- Applied star schema for optimized performance
### 3. DAX Calculations
Built measures like:
- Total Orders
- On-Time Delivery Rate
- CSAT %
- Avg Delivery Time
- Dynamic KPIs
### 4. Dashboard Design
- Designed interactive and user-friendly visuals
- Used slicers for Year and Month filtering
- Applied consistent color themes and layout

## 📊 Dahboard Features

### 🔹 Key KPIs
- Total Orders
- On-Time Delivery Rate
- Customer Satisfaction (CSAT %)
- Average Delivery Time

### 🔹 Driver Analysis
- Experience vs Ratings scatter analysis
- Drivers with highest delays
- Individual driver performance insights

### 🔹 Hubs Analysis
- Hub capacity vs processed orders
- Hub performance ranking
- Processing efficiency heatmap

### 🔹 Vehicles Analysis
- Active vs Maintenance vehicles
- Orders by vehicle model
- Vehicle type distribution
- Vehicle age vs breakdown trends
